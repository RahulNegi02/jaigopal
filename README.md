
# jaigopal
Simple Node.js + React single-page application.

Run the server:

```powershell
npm start
# or
node index.js
```

The server prints the URL it is listening on and will try the next port if 3000 is in use.

Open the printed URL in your browser (for example http://localhost:3000).

Vercel deployment:

- The app is ready for static deployment from the project root.
- The [vercel.json](vercel.json) file provides SPA routing support.
